# Employee Attrition Analysis

## 📌 Project Overview
This project analyzes employee attrition data to identify the key factors influencing employee turnover. The analysis includes data cleaning, exploratory data analysis (EDA), and visualizations to uncover meaningful business insights that can help improve employee retention.

---

## 📂 Dataset
- Employee Attrition Dataset
- Cleaned and analyzed using Python

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Data Visualization

---

## 📈 Key Insights
- Sales department recorded the highest employee attrition.
- Employees who worked overtime were more likely to leave the company.
- Employees with lower monthly income showed higher attrition.
- Monthly income generally increased with total working experience.
- Most employees stayed with the company, but attrition patterns varied across departments and job roles.
